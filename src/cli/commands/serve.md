### Usage:

```
docsense serve [Options]

Serves up a local docsense app that watches for changes

Options:
  --out          directory docs are generated in, when thse
                 change the browser will refresh

  --glob         files matching this glob pattern will
                 trigger a rebuild      [default: ["**/*"]]

  --root         root path to watch for file changes that
                 trigger a new doc build

  -v, --version  Show version number              [boolean]
  -h, --help     Show help                        [boolean]

```
