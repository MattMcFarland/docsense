# `docsense init`
## Usage:

```
Initializes docsense config file (.docsenserc)

Options:
  --dir          directory to add rc file    [default: "."]
  --out          where docs will be written when using
                 docsense build         [default: "./docs"]
  --main         main markdown file (without .md extension)
                                      [default: "./README"]
  --files        files to generate documentation on (can be
                 glob patterns)[default: ["./src/**/*.js"]]
  -v, --version  Show version number              [boolean]
  -h, --help     Show help                        [boolean]
```
